生成type声明
在 src\typings\app.d.ts 文件生成声明 User 以及 form 子属性

注意：
需要将首字符转为小写
title:string; 为固定的每次生成都需要且保持在第一个位置
代码位置插入在 manage: {} 的上方不是在 manage内
示例
xxx {
  title: string;
  field：string
  .....
  form: {
    field：string
    .....
  }
}
